Official Game Guide

Introduction

Side Scroller Sprint is a high-intensity, endless runner arcade game developed in HTML5. Designed to test reaction time and precision, the game strips away complex mechanics to focus on one core skill: timing.

How to Play

  • Desktop: Press SPACEBAR or UP ARROW to jump.
  • Mobile: Tap anywhere on the screen.

Mechanics & Scoring

Every 500 frames, the game speed increases. You earn 1 point for every obstacle cleared. Global high scores are saved via Firebase.

Built with Vanilla JavaScript and HTML5 Canvas.
